Smart Tender Evaluation Platform

 Overview :-

The Smart Tender Evaluation Platform is a web-based application designed to automate and simplify the evaluation of tenders using AI. It helps organizations compare bids efficiently, reduce manual effort, and make data-driven decisions.
The system leverages modern web technologies and AI integration to analyze tender documents and provide intelligent insights.

 Features :-

~ Upload and manage tender documents
 
~  AI-based evaluation using Gemini API

~ Automated scoring and comparison of bids

~ Easy filtering and selection of best proposals

~ Responsive and user-friendly UI

~ Fast and scalable architecture

 Tech Stack :-

~ Frontend:
React.js
TypeScript
Vite
CSS

~ Backend:
Node.js
Express.js

~ AI Integration:
Gemini API (Google AI)

 Usage :-
 
~ Upload tender documents

~ Let the AI analyze bids

~View evaluation results

~Compare and select the best proposal
